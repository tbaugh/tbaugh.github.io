I've completed my first project with this blog, adding Bootstrap. Boostrap is a front-end framework that makes it really easy to rapidly layout and style websites. I've touched Boostrap a couple times in doing Rails tutorials, but this was the first time I've tried using it with a different sort of project and without following a set tutorial.

However that isn't to say that I didn't use some instructions! Because of some funky stuff with the way Jekyll works on Github we aren't able to install Bootstrap the normal way. I found this blog post written by Github user veithen to be quite helpful. You use the Sass port rather than a traditional installation.

This little project was pretty easy but definitely required a little bit of trouble shooting and Googling to get through. Now to learn more about working with Bootstrap.
